# Tabs

Organises content into tabbed panels. Only one panel is visible at a time; the user switches panels by clicking the tab labels.

## Default

| Tabs                    |                              |
|-------------------------|------------------------------|
| Label *(required)*      | Content *(required)*         |
| Tab label text          | Any block content — text, images, nested blocks |

Each authored row becomes one tab. The first column is the tab label; the second column is the panel content.

## Variations

### Outlined

| Tabs (outlined)         |                              |
|-------------------------|------------------------------|
| Label *(required)*      | Content *(required)*         |
| Tab label text          | Any block content            |

Tab buttons render with a border rather than a filled active indicator.
