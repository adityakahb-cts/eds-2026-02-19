# Card

Displays a bordered card with a top image, a headline, and a CTA link.

## Default

| Card                       |
|----------------------------|
| Image *(required)*         |
| `<picture>` element rendered at the top of the card |
| Heading *(required)*       |
| Headline text shown below the image                  |
| CTA *(required)*           |
| Link styled as a text CTA with a trailing arrow      |
